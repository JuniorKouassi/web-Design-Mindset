## Updating the founder photo

1. Drop new portrait at `img/founder-original.jpg`
2. Run:
   ```bash
   node scripts/process-photo.js
   ```
3. Script auto-generates `founder-duotone.png` and `founder-duotone@2x.png`
4. Refresh — no HTML changes needed (files keep the same name)

> **Tip:** Photos with a pure black background work best — the black becomes fully transparent on the site via `mix-blend-mode: screen`.

## Running locally

Right-click `index.html` → *Open with Live Server*

```bash
cd "WEB DESIGN MINDSET"
npx serve .
# then open http://localhost:3000
```

## Languages (EN / FR)

All text is driven by `js/i18n.js`. To edit a translation:
1. Open `js/i18n.js`
2. Find the key under `en:` or `fr:`
3. Edit the string — changes apply instantly on refresh

## Font swap (optional)

Current fonts (Google Fonts, free):
- **Display italic:** DM Serif Display
- **Body/UI:** Inter 400/500/600

To upgrade to **PP Editorial New** + **PP Neue Montreal**:
1. Remove the Google Fonts `<link>` from each `.html` file
2. Add `@font-face` rules in `css/styles.css`
3. Update the CSS variables:
   ```css
   --font-display: 'PP Editorial New', Georgia, serif;
   --font-body:    'PP Neue Montreal', system-ui, sans-serif;
   ```
